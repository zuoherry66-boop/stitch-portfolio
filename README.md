# Stitch Portfolio

可直接打开 `index.html` 预览。

后续你主要改这几个位置：

- 首页大标题、介绍文案：`index.html` 里的 `.hero-copy`
- 案例卡片标题与说明：`index.html` 里的 `.featured-grid`
- 分类名称：`index.html` 里的 `.category-strip`
- 联系方式与社媒：`index.html` 里的 `.site-footer`
- 整体颜色、圆角、字体：`styles.css` 顶部 `:root`

当前已经做好：

- 桌面端与移动端响应式布局
- 手机端折叠菜单
- 案例分类筛选
- 预留上传入口与联系区
